Python - Password Checker

There are many Passwords out there - many weak ones.
The goal with this script was to enter some potential passwords.
Then the script should check wheter its secure or not.

It follows several guidelines:
- has the password more than 8 characters ?
- has the password more than 2 numbers ?
- has the password more than 1 special character ?
Only if this tests were sucessful - the passwors is rated as secure.
